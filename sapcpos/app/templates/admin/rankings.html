⚙️ Algorithm: QuickSort

Students are ranked using the QuickSort algorithm (O(n log n) average time complexity). The algorithm partitions the student list around a pivot GPA value, recursively sorts each partition, and produces a fully sorted ranking. On the Philippine grading scale, 1.0 is the highest (Excellent) and 5.0 is failing, so rank #1 is the student with the lowest GPA number.
